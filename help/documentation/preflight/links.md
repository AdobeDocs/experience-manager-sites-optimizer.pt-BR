# Oportunidade de Links de comprovação

![Oportunidade de Links de Simulação](./assets/links/hero.png){align="center"}

A oportunidade Links de comprovação no Sites Optimizer ajuda a identificar e solucionar problemas relacionados ao uso de links no seu site. Essa oportunidade se concentra na avaliação da estrutura de links do site e fornece recomendações acionáveis para melhoria.
