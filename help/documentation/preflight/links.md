# Oportunidade de simulação de links

![Oportunidade de simulação de links](./assets/links/hero.png){align="center"}

A Oportunidade de simulação de links no Sites Optimizer ajuda a identificar e solucionar problemas relacionados ao uso de links no site. Essa oportunidade se concentra na avaliação da estrutura de links do site e fornece recomendações práticas para melhoria.
